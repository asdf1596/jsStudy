# jsStudy

# 콘솔 객체

-   브라우저의 디버깅 콘솔에 접근할 수 있는 메소드를 제공을 한다.

# var,let,const

-   var:중복 선언과 재할당 가능

-   let:중복 선언 불가, 재할당 가능

-   const:중복 선언 불가하고 재할당도 불가능

## var

-   변수 선언이 자유롭지만 오히려 그로 인해 코드가 길어지고 많이 재할당되면 값을 파악하기 힘들어 질 수 있다.

## let

-   중복 선언 불가하지만 재할당(값 변경)은 가능하다.

## const

-   중복 선언과 재할당 모두 불가능.

# scope(유효한 참조 범위), 호이스팅

## scope

-   var:함수 범위 스코프

-   let,const:블록 레벨 스코프
